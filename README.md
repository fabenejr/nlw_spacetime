# Uma cápsula do tempo desenvolvida para registrar suas lembranças! 

Memories é um software para registro de lembranças, disponível em versão web e mobile. 

### Bibliotecas:
- TypeScript  
- Prisma


### Frameworks:
- Next.js   
- Fastify
- React
- React Native
- Expo
